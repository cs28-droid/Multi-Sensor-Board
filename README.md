# Multi-Sensor-Board

This is a Multi Sensor Board built using Arduino. This is an all in one sensing system built using 5 sensors.

Sensors used in the project:-
-> MQ-2 gas sensor
-> Ultrasonic sensor
-> Proximity sensor
-> DHT11 temperature + humidity sensor

Other Hardware:-
-> Arduino Uno
-> Buzzer
-> LEDs
-> 220 ohms resistors
